Flexbox: En forma normal, los objetos tienen un 

display flex; indicará que sus hijos serán "flexibles".

flex-direction row/column: elegir dirección
flex-wrap wrap/nowrap: multilinea cuando llegue al limite, con wrap se distribuyen mientras se agregan elementos, con nowrap se aplastan sin desplazarse, se angostan para ocupar el espacio de una linea, se comprimen las cajas

flex-flow: abreviacion de flex-direction y flex-wrap
justify-content: alinear horizontalmente a los hijos(row) o verticalmente (column)
align-items: para alinear de forma vertical
gap: espacio fijo entre los contenedores, pero solo sirve si esta activado wrap
calc(): operaciones en parametros de css para medidas
overflow: evita que el contenido se desborde del contenedor

Diseño Responsivo (responsive design): se refiere a la idea de que un sitio web debería mostrarse igualmente bien en todo tipo de dispositivo, desde monitores de pantalla panorámica hasta teléfonos móviles. Es importante empezar el diseño a partir de vista de movil, e ir escalando a tablets y escritorio, buena práctica.

Breakpoints: Categorías, casi estándares para definir lo que es cada dispositivo a traves de su medida en pixeles:
320px: Para dispositivos con pantallas pequeñas, como telefonos en modo vertical
480px:  para dispositivos con pantallas pequeñas, como telefonos en modo horizontal

Mobile first:
@media (min-width: ...){...}
Mobile first ------------------> Desktop last

.miestilo{
	background-color: green;
}
@media (max-width: 480px) {
	.miestilo{
		background-color: red;
	}
}
